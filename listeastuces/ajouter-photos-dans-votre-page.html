Comment modifier votre page facebook ?

Pour modifier votre page maintenant, nous allons tout simplement sur l'application Facebook, connecter vous.

  • Aller dans le menu de votre profile

  • Défiler vers le bas et cliquer sur page

  • Cliquer sur votre page

  • Dans la bannière de votre page, cliquer sur l'icône camera

  • Cliquer sur importer une photo

  • Choisit votre photo à importer dans votre galerie ,puis enregistre

Pour ajouter un logo, c'est la même chose, pas besoin de vous expliquer, vous le verrez dans la vidéo au-dessus.

Bravo !! vous avez créé votre page Facebook avec succès et tout seule.
