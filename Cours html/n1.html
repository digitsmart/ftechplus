Bienvenue dans le premier chapitre du cours html et css sur android, dans ce chapitre nous allons voir ensemble comment gérer un site web avant même de le construire.


Comment créer mon premier projet site web sur android ?
C’est pas trop compliquer pour créer un site a parti de votre android, il suffit d’avoir des logiciel qui est capable de comprendre le langage html et css.

Avant de commencer, il faut savoir qu’il existe deux types de site :
Site statique : Une page web statique est une page web dont le contenu ne varie pas en fonction des caractéristiques de la demande, c'est-à-dire qu'à un moment donné tous les internautes qui demandent la page reçoivent le même contenu.

Site dynamique : Une page web dynamique est une page web générée à la demande, par opposition à une page web statique.

Le contenu d'une page web dynamique peut donc varier en fonction d'informations (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur, etc.) qui ne sont connues qu'au moment de sa consultation. À l'inverse, le contenu d'une page web statique est a priori identique à chaque consultation.

tout d'abord, avant de commencer nous allons voir quelques definition tres inportant, qui vous permettera de bien gerer votre projet web site sur android.


C'est quoi un dossier?
Un dossier informatique, est un repertoire contenant des sous-dossiers et des fichiers.


C'est quoi un fichier?
Un fichier est un enssemble de donnees considerees comme unite. Il peut de tout type: un document texte. un logiciel, une image etc... chaque fichir porte un nom suivi d'une "extension".


une extension est un suffixe de nom de fichier fait pour identifier son format.

Et bien pour créer une page web, il ne suffit pas de taper simplement du texte, comme on vient de le faire. En plus du texte, il faut utiliser des balises, qui vont donner des instructions a l’ordinateur comme « allez à la ligne ».

Squellette minimal par defaut d’une page html

Donc, nous allons allons voir les balises qui compose l’esquellette par defaut d’une page html et il sont tres important, car sans eux vous ne pourrex pas créer votre page web.


Voici la structure minimale d’une page HTML 5 :

  • <html>
    • <head>
    • </head>
    • <body>

    • </body>
  • </html>
    •

La balise <html> matérialise le début du document HTML, et la balise <html> la fin de ce document.

Les balises <head> et </head> délimitent l’en-tête du document. Vous verrez ce qui peut être placé entre ces deux balises dans une autre rubrique.

Enfin, les balises <body> et </body> délimitent le corps du document. Tout ce qui est écrit entre ces deux balises apparaîtra dans le navigateur Web.
