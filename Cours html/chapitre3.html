Dans ce chapitre nous allons découvrir les bases du langage html et enregistrer notre premier projet page web site sur Android, dans la suite de ce cours, je travaillerai avec anWriter free html , un éditeur de texte pour Android.

Dans votre éditeur de texte écrit ce qui vous passe à la tête ou vous pouvez écrire les mêmes phrases que moi.

EX : Bonjour, mes visiteurs, j’espère que vous bien comprendre ce cours Ce cours est valable pour tous les débutants en html et du css.

Maintenant, enregistrons ce fichier. Pour cela c’est très simple, comme dans tous les programmes, vous avez un menu, cliquer sur l’icône « save » en haut de votre application et une boite de dialogue s’ouvre pour vous demander ou enregistrer le fichier et sous quel nom.

Enregistrer le ou vous voulez, donnez au fichier un nom en terminant par .html. EX : index.html ( index est le nom que j’aie choisi pour mon fichier et le ,html est l’extension par défaut du langage html.

Je vous recommande d’enregistrer votre fichier dans votre dossier principal, que vous avez créé pour votre site web. Apres l’enregistrement aller dans « mes fichiers »de votre téléphone et ouvrir votre dossier principal et vous verrez votre fichier. L’icône qui représente le fichier dépend de votre navigateur web par défaut en général.

Maintenant, pour visualiser votre travail, cliquer sur le fichier que vous avez enregistré dans votre dossier, on va vous proposer de l’ouvrir avec un éditeur de texte ou un navigateur, choisit le navigateur pour le visualiser sur votre Android.

Que se passe-t-il ? on dirait que tout le texte s’affiche sur la même ligne, alors qu’on avait écrit deux lignes de texte différentes ?

Et bien pour créer une page web, il ne suffit pas de taper simplement du texte, comme on vient de le faire. En plus du texte, il faut utiliser des balises, qui vont donner des instructions a l’ordinateur comme « allez à la ligne ».
